# Трекер Прочитанных Книг
## Позволяет Консольное приложение для учёта прочитанных книг с модульной архитектурой

# Модули:
– main.py: Точка входа, пользовательский интерфейс и инициализация других модулей
– storage.py: Загрузка/выгрузка данных в файл и добавление/удаление данных
– stats.py: Отображение статистики и расчёт средней оценки
– models.py: Класс Book()

Для запуска введите в терминал python main.py

Пример работы:

Введите команду:
– 1. Добавить книгу
– 2. Показать все книги
– 3. Показать среднюю оценку
4. Статистика по авторам
5. Удалить книгу
6. Выход
